# Solaris Peruvian ☀️

Proyecto web de **Soleil Style**. Una plataforma moderna desarrollada con Node.js y Express.

* HTML, CSS y JavaScript
* Node.js y Express

## 🚀 Cómo abrir el proyecto
1. Instalar archivos: `npm install`
2. Iniciar: `node server.js`
3. Ver en el navegador: `http://localhost:3000`
