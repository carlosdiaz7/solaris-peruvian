## 🚀 Guía de Inicio rápido

1. **Instalar librerías:** Ejecuta `npm install` en la terminal (esto descarga la carpeta `node_modules`).
2. **Arrancar:** Ejecuta `node server.js`.
3. **Ver sitio:** Entra a `http://localhost:3000` en tu navegador.
